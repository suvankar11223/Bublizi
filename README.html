Features
Real-time Messaging
Instant delivery via Socket.IO with typing indicators and read receipts.
Voice & Video Calls
WebRTC with TURN relay, ICE negotiation, and renegotiation support.
AI Chat Assistant
Built-in bot with context-aware suggestions and cross-chat linking.
Voice Messages
Record and send voice notes inline with playback support.
File Sharing
Cloudinary-backed image and file uploads with progress tracking.
Push Notifications
Firebase Cloud Messaging for reliable background alerts.
Contact Sync
Automatic phone contact discovery to find existing users.
Online Presence
Redis-backed real-time user status at any scale.
Message Reactions
Emoji reactions and message pinning for important threads.
